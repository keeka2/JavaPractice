CBD - 모든 데이터를 component화 시킴

Bean - 하나의 컴포넌트(class)

Container - 무엇을 담고 있는 것
Component - 모든 데이터를 부품화
Container에 Component가 있음

DAO / DBBEAN - 데이터를 저장, 수정,읽기...만 하는 클래스
DTO / VO(value Object) / DataBean - 값을 넣고 빼는(Get Set - 캡슐화되있는 클래스)

=> 얘네들을 편하게 관리 가능하게 해주는 것(프레임워크)

자바 컴파일러 - JVM, JIT(메모리를 실시간으로 만들어 줌)-C언어는 처음에 만들어짐
JIT+절차지향: 쿠팡

frameWork
-틀을 갖다 조립해서 쓰즈아
- 관점지향
- 절차지향, 객체지향으로도 관점지향이 가능함
- ex) 배/사과 => 배를 갖다 쓰는 사람은 배를 먹는 배, 타는 배 ...으로 쓸 수 있음
- 인터페이스를 쓰면 표준화가 가능함.
